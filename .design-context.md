# Design Context — FREDDY EPR Platform

## Users
- 中国制造企业出口经理、跨境电商卖家、合规专员
- 场景：办公室PC白天使用为主，手机为辅
- 核心任务：了解合规要求 → 在线签约 → 下载合同 → 上传签章 → 跟踪状态
- 情感目标：安心（合规有保障）、信任（德国本土授权代表）、高效（5分钟完成）

## Brand Personality
- **三个词**：严谨 · 可信赖 · 专业
- **品牌调性**：中德双主体，德国工程精密感 + 中国服务务实感
- **差异化**：不是SaaS创业公司，是高端跨境合规咨询平台
- **参考**：德国工业品牌（Bosch、TÜV）+ 高端咨询公司

## Aesthetic Direction
- **配色**：深蓝 (#0a1628 → #1e3a5f) + 白色为主，金色点缀
- **主题**：浅色为主（白底），深蓝用于Hero和关键模块
- **风格**：精致专业，德国工业精密感，克制留白
- **反参考**：AI生成的暗色+霓虹光效、玻璃态卡片、渐变文字

## Design Principles
1. **信任即设计**：留白、对称、稳重配色传递可靠感
2. **精密而非冰冷**：金色微交互带来暖意
3. **信息有出处**：数字标注来源，条款引用法规
4. **中文呼吸感**：充足行间距，避免信息拥挤
